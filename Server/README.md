# Утилита управления Tech-системой
Консольная программа для тестирования и управления оборудованием (MA/FURKE) на новом БОС(миландры).

## Использование
./program [ОПЦИИ](2 ключа) РЕЖИМ(работы)

## Опции

| Опция | Описание |
|-------|----------|
| `-MA` | Использовать конфигурацию MA |
| `-FURKE` | Использовать конфигурацию FURKE |
| `-U`, `--udp` | UDP сокет для связи с VIOD |
| `-D`, `--dspapi` | DSP API для связи с VIOD |
| `-h`, `--help` | Справка |

## Режимы работы

| Режим | Действие |
|-------|----------|
| `-t`, `--test` | Запуск теста AF08 |
| `-l`, `--load-css` | Загрузка CSS |
| `-s`, `--send-upr` | Отправка UPR в CSS |
| `-f`, `--full-cycle` | Полный цикл (SUM & CSS) |
| `-i`, `--init-regs` | Инициализация регистров |

## Примеры

```bash
# Запуск теста с конфигурацией MA
./program -MA -U -t

# Отправка UPR через UDP
./program -FURKE -U -s

# Полный цикл с DSP API
./program -MA -D -f
